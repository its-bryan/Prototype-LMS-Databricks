# Hertz Visual & Chart Style Guide

## 1. Core Brand Colors

### Primary
- **Hertz Yellow (Primary Highlight):** `#F5C400`
- **Charcoal / Near-Black (Text, Axes):** `#1A1A1A`
- **White (Background):** `#FFFFFF`

### Secondary / Utility
- **Dark Gray (Secondary Series):** `#6E6E6E`
- **Light Gray (Gridlines / Baselines):** `#E6E6E6`
- **Positive Green (Improvements):** `#2E7D32`
- **Negative Red (Declines):** `#C62828`

---

## 2. Chart Design Principles

- White background only
- No gradients, shadows, or borders
- Minimal gridlines (horizontal only)
- Strong visual hierarchy: **yellow used sparingly and intentionally**
- Never have overlapping elements (e.g. titles, legends)

---

## 3. Chart-Specific Rules

### Bar Charts
- Primary metric → **Hertz Yellow**
- Comparison / prior period → **Dark Gray**
- No bar outlines

### Dual-Axis Bar + Line Charts
- Bars (volume / $) → **Hertz Yellow**
- Line (rate / %, efficiency) → **Charcoal or Dark Gray**
- Line thinner than bars
- Line markers small, filled
- Secondary axis labels match line color (never yellow)

### Multi-Series (3+ Series)
1. Primary → `#F5C400`
2. Secondary → `#6E6E6E`
3. Tertiary → `#BDBDBD`

Avoid non-brand colors (blue, purple, teal, etc.).

---

## 4. Axes, Labels & Gridlines

- Axis text: `#4A4A4A` or `#1A1A1A`
- Axis lines: removed or very subtle
- Gridlines: `#E6E6E6`, horizontal only
- Chart background: white

---

## 5. Text & Annotation Color Usage

- Chart titles: Charcoal / Black
- Subtitles: Dark gray
- Key callouts / headline numbers: Yellow or black
- Footnotes / sources: Light gray

---

## 6. Make sure chart elements, titles, and legends do not overlap. Give enough space
